## What is Sorionlib?

Sorionlib is a powerful JavaScript/TypeScript library designed to simplify common development tasks. It provides a unified toolkit for building applications with features like API clients, Discord bot tools, database helpers, and core utilities.

## Purpose

Sorionlib aims to reduce boilerplate code and provide consistent, well-tested utilities that developers can rely on across projects. Instead of reinventing the wheel for common tasks, you can focus on building your application's unique features.

## Key features

- **Core utilities** - Common helper functions for strings, arrays, objects, and more
- **API client** - Built-in HTTP client with authentication and error handling
- **Discord tools** - Command builder, event system, and middleware for Discord bots
- **Database helpers** - Simplified connections and CRUD operations for MongoDB and MariaDB
- **Error handling** - Custom error classes with detailed error codes

## Why use Sorionlib?

- **Simplicity** - Clean, intuitive API that's easy to learn
- **Modularity** - Use only what you need
- **TypeScript support** - Full type definitions included
- **Well documented** - Comprehensive guides and examples
- **Active maintenance** - Regular updates and bug fixes

## Hello World

Get started with Sorionlib in just a few lines:

```js
import { Sorion } from "sorionlib";

const client = new Sorion();
client.start();

console.log("Hello from Sorionlib!");
```
